# PC Cleaner Utility

Консольная утилита для обслуживания Windows-ПК: диагностика, очистка, проверка подозрительных объектов, управление автозагрузкой и набор системных твиков.

## Возможности

- Сводная информация о системе (CPU, RAM, диск, аптайм, сеть)
- Очистка временных файлов и системных кэшей
- Базовый threat scan (файлы, процессы, hosts, автозагрузка)
- Запуск Windows Defender quick scan
- Управление автозагрузкой через реестр
- Сетевые инструменты (flush/release/renew/ping/traceroute)
- Performance tweaks (визуальные эффекты, DNS, telemetry и др.)
- Поиск дубликатов файлов (MD5)
- Поиск крупных файлов
- Вывод списка установленных программ

## Требования

- Windows 10/11
- Python 3.9+
- Права администратора для части функций

## Быстрый старт

```powershell
python pc_cleaner.py
```

Если запуск без прав администратора, утилита предложит перезапуск с elevation.

## Важные замечания

- Скрипт удаляет файлы только в пользовательских/системных временных локациях и кэшах.
- Некоторые операции (например, `sfc`, `dism`, изменение реестра) потенциально долгие и требуют осознанного запуска.
- Перед массовыми твиками рекомендуется создать точку восстановления системы.
